# E-Commerce Frontend (React)

## Installation

```bash
npm install
```

## Configuration

Le fichier `.env` doit contenir :
```
VITE_API_URL=http://localhost:3000
```

## Prérequis
- Le backend doit être démarré sur le port 3000
- Node.js 18+

## Démarrage

```bash
npm run dev
```

L'application sera disponible sur `http://localhost:5173` (ou port alternatif)

## Fonctionnalités
- Affichage des produits
- Interface d'administration
- Gestion du panier
- Authentification

## Technologies
- React 18
- TypeScript
- Vite
- CSS moderne
